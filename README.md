# Projeto Web Completo

Este projeto foi desenvolvido durante as aulas do **Projeto AlphaEdtech**, em **abril de 2026**, com o objetivo de praticar e consolidar conhecimentos fundamentais de desenvolvimento web utilizando **HTML5** e **CSS3**.

## Sobre o Projeto

A proposta consistiu na criação de uma página web completa, reunindo diferentes elementos estruturais e visuais estudados em aula.

O projeto contém:

- Listas não ordenadas
- Listas ordenadas
- Lista de definição
- Tabelas simples
- Tabelas avançadas com células mescladas
- Formulários interativos
- Uso de Classes e IDs
- Estilização com CSS
- Organização semântica da página

## Tecnologias Utilizadas

- HTML5
- CSS3
- Visual Studio Code
- Git
- GitHub

## Objetivo

Aplicar na prática os principais conceitos introdutórios de desenvolvimento front-end, estruturação de páginas web e estilização visual.
